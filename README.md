# Save-it

개인 메모 및 북마크 관리 웹앱

## 주요 기능

- **메모 저장**: 제목, 메모 내용, URL, 폴더(카테고리) 정보를 입력해 메모를 저장할 수 있습니다.
- **폴더 분류**: 메모를 폴더별로 분류할 수 있으며, 폴더를 새로 만들거나 기존 폴더를 선택할 수 있습니다.
- **폴더별 보기**: 상단의 폴더 선택 드롭다운을 통해 특정 폴더에 속한 메모만 필터링해서 볼 수 있습니다.
- **검색**: 메모의 제목, 내용, URL을 실시간으로 검색할 수 있습니다.
- **메모 삭제**: 개별 메모 삭제, 선택 삭제, 전체 삭제가 가능합니다.
- **다중 선택**: 메모를 여러 개 선택해 한 번에 삭제할 수 있습니다.
- **다크/라이트 모드**: 상단 버튼으로 테마를 전환할 수 있습니다.
- **로컬스토리지 저장**: 모든 데이터는 브라우저 로컬스토리지에 저장되어 새로고침해도 유지됩니다.

## UI/UX 특징

- 폴더 선택 드롭다운 앞에 폴더 아이콘이 있어 폴더임을 직관적으로 알 수 있습니다.
- 폴더 선택 시 새 폴더명 입력란이 비활성화되고, 배경색이 어둡게 처리되어 구분이 명확합니다.
- 메모의 폴더 태그는 제목 위에 표시되며, 제목과 메모 내용 왼쪽에는 불필요한 아이콘이 없습니다.
- 전체 선택 텍스트를 클릭하거나 키보드로도 전체 선택/해제가 가능합니다.
- 전체 배경이 흰색으로 깔끔하게 구성되어 있습니다.

## 폴더/메모 입력 방법

1. 상단 입력 폼에서 폴더를 선택하거나 새 폴더명을 입력합니다.
2. 제목, URL, 메모 내용을 입력하고 저장 버튼을 누르면 메모가 추가됩니다.
3. 폴더를 선택하면 해당 폴더의 메모만 볼 수 있습니다.

## 기술 스택

- React

- Tailwind CSS
- Lucide React (아이콘)
